# simp-win-video-player 视频播放器
 一个使用Electron.js的Windows视频播放器

## 下载

前往 [Releases](https://github.com/haoqi75/simp-win-video-player/releases) 下载最新的安装包。

## 功能

- 支持 MP4、MKV、AVI 等常见视频格式。
- 通过文件选择对话框选择本地视频文件。

## 运行

1. 下载并安装应用。
2. 打开应用后，点击“选择视频文件”按钮加载视频。
3. 使用内置控件播放视频。
